## 2024-03-21 - Added Keyboard Shortcut Hints to Go TUI
**Learning:** Terminal User Interfaces (TUIs) built with bubbletea often lack discoverable quit commands, which can leave users feeling trapped.
**Action:** Always ensure interactive TUI components have explicit keyboard shortcut hints (like 'Press q or ctrl+c to quit') visible to the user. Style these hints using `charmbracelet/lipgloss` so they are helpful but not distracting.
