## 2024-05-18 - TUI Exit Hints
**Learning:** Terminal User Interfaces (TUI) without explicit, constantly visible exit commands (like 'Press q to quit') cause significant user anxiety as they feel "trapped" in the alternate screen buffer, unlike web apps where the browser back button or closing the tab provides an obvious escape hatch.
**Action:** Always include a styled, subtle but visible exit hint in the main view of any charmbracelet/bubbletea application.
