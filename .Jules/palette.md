## 2026-05-02 - Prevent TUI User Trapping
**Learning:** Users can easily feel trapped or frustrated in terminal user interfaces (TUIs) when exit commands are not explicitly visible, as conventions (like Ctrl+C or 'q') aren't universal to all users.
**Action:** Always provide explicitly styled, visible keyboard exit paths (like 'Press q or ctrl+c to quit') in TUI applications to maintain a baseline of accessibility and user control.
