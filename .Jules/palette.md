## 2026-04-30 - TUI Keyboard Escape Paths
**Learning:** In terminal user interfaces (TUIs), users experience "terminal trap anxiety" when clear exit or navigation paths aren't immediately visible, contrasting with web apps where the browser provides a persistent back/close button.
**Action:** Establish a reusable TUI pattern for Cantor: explicitly display keyboard shortcut hints (e.g., 'Press q or ctrl+c to quit') at the bottom of views using subtle contrasting colors (like lipgloss #E0D4FF) to provide safety without distracting from the main actuation console.
