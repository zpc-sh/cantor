## 2024-05-24 - TUI Keyboard Traps
**Learning:** Users can feel trapped in full-screen terminal applications (like alternate screen TUIs) if keyboard shortcuts are not explicitly visible, leading to frustration and panic quitting.
**Action:** Always ensure interactive TUI components include explicit, persistently visible keyboard shortcut hints (e.g., 'Press q or ctrl+c to quit') styled distinctly from the main content.
