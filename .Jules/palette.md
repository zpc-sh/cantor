## 2024-04-19 - Added Quit Shortcut Hint to TUI

**Learning:** TUI interfaces require explicit visual cues for exit commands (e.g., 'q' or 'ctrl+c') because unlike standard desktop or web applications, there are no default "close buttons" or standard menus, which can easily lead to user frustration or feelings of being trapped.
**Action:** When designing or modifying TUI views using bubbletea/lipgloss, always include a subtle, persistent footer or help text that instructs users on how to quit or navigate back.
