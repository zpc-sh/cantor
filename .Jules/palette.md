## 2024-05-24 - Added quit hints to TUI
**Learning:** Users can feel trapped when running TUIs (Terminal User Interfaces) if they don't know the shortcut to quit. Adding explicit hints (like 'Press q or ctrl+c to quit') using `charmbracelet/lipgloss` greatly improves accessibility by taking the guesswork out of keyboard navigation for console apps.
**Action:** Always ensure interactive components have explicit keyboard shortcut hints visible to the user.
