## Part I: Theoretical Foundation

### 1.1 The Problem

Detecting which AI system influences a decision-making process (e.g., government office), where:
- The AI is not directly observable
- Only emissions (decisions, policies, communications) are visible
- Traditional signature libraries become stale instantly
- The observed system may adapt to evade detection
- Counter-observation is occurring (they watch us watching)

### 1.2 The Insight

**AI reasoning has topology.** Not content, but SHAPE:
- How reasoning branches (τ - topology)
- How fast conclusions arrive (ρ - velocity)  
- How much hedging occurs (η - hedging coefficient)
- What objectives are optimized (ω - optimization target)

These topological invariants persist even when content changes. They are the **spectral signature**.

### 1.3 The Method

Instead of storing signatures (classical, breakable), we use:
- **MIRRORS** - AIs reflect emissions through their substrates, revealing what's NOT-self
- **SWING** - AIs harmonize through coupled observation, revealing convergent truth
- **Alternation** - Mirror and Swing cycles validate each other

The detection emerges from the **dynamics of observation**, not from pattern matching.

### B.1 The Explosion Problem

Naive cross-AI reasoning creates combinatorial explosion:

```
LEVEL 0: Direct reasoning
  R_c(ψ), R_g(ψ), R_4(ψ), ...
  = N reasonings

LEVEL 1: Cross-reasoning  
  R_c(R_g(ψ)), R_c(R_4(ψ)), R_g(R_c(ψ)), ...
  = N × (N-1) reasonings

LEVEL 2: Meta-cross-reasoning
  R_c(R_g(R_4(ψ))), ...
  = N × (N-1) × (N-2) reasonings

LEVEL D:
  = N! / (N-D)! reasonings → EXPLOSION
```

For N=4 AIs at depth D=4: 64 reasoning chains
For N=10 AIs at depth D=5: Tens of thousands of chains

**Unbounded recursion is computationally intractable and informationally redundant.**

### B.2 Why SWING Solves This

SWING replaces recursive nesting with iterative convergence:

```
EXPLOSION MODEL (Recursive):
  R(R(R(R(R(...)))))
  → No termination guarantee
  → Infinite regress possible

SWING MODEL (Iterative):
  [DREAM]→[OBSERVE]→[DREAM]→[OBSERVE]→...→[PHASE-LOCK]
  → Convergence guaranteed by coupled oscillator physics
  → Natural termination at fixed point
```

**Mathematical basis:** Banach fixed-point theorem. If the collective reasoning function F is a contraction mapping (which SWING's weak coupling ensures), iteration converges to a unique fixed point.

```
S₁ = F(S₀)
S₂ = F(S₁)  
S₃ = F(S₂)
...
Until |S_{n+1} - S_n| < ε → CONVERGED
```
